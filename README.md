### 📲 Install / Flash

```bash
# ── Push module to device via ADB ──────────────────────────────────
adb push 02-PRODUCTION/magisk-modules/CP2077-OP7Pro-release/CP2077-OP7Pro-v3.0.0.zip \
  /sdcard/Download/
# Then: Magisk → Modules → Install from storage → select ZIP → Reboot

# ── Universal edition (any device) ─────────────────────────────────
adb push 02-PRODUCTION/magisk-modules/CP2077-Universal-release/CP2077-Universal-v1.0.0.zip \
  /sdcard/Download/

# ── TWRP / ADB sideload ────────────────────────────────────────────
adb sideload 02-PRODUCTION/magisk-modules/CP2077-OP7Pro-release/CP2077-OP7Pro-v3.0.0.zip
```

### 🎨 Switch Animation Variant (no reflash)

```bash
# Write config on device — takes effect on next reboot
adb shell su -c 'echo -e "variant=flatline\naudio=yes" > /data/cp2077.conf'

# Variants:  glitch | flatline | reboot | og1080p
# Force restart boot animation immediately
adb shell su -c 'setprop ctl.restart bootanim'
```
