# Changelog — CP2077 Universal Edition

## v1.0.0 (2026-05-13)
### Initial Release
- Universal build supporting ALL Android devices
- Auto-detects ROM family (14 supported: AOSP, LineageOS, Samsung One UI, OOS/ColorOS, MIUI/HyperOS, Pixel, CalyxOS, GrapheneOS, crDroid, Evolution X, yaap, ArrowOS, PixelOS, RisingOS, DerpFest)
- Auto-detects root solution (Magisk 20.4+, KernelSU, APatch)
- 4 boot animation variants at canonical 1440×3120 (auto-scaled on device): CyberGlitch, Flatline, Re-Boot, Original
- Matching shutdown animations for all 4 variants
- CP2077 audio pack (7 OGG files)
- Resolution install path matrix covers all major ROM families
- Min API 21 (Android 5.0+)
