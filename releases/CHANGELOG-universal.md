### ⚡ Key Technical Features

| ✅ Feature | 📋 Detail |
|:----------|:---------|
| Auto-detect ROM | `getprop` fingerprinting at install time — 15 ROM families |
| Auto-detect root | Magisk 20.4+ · KernelSU · APatch |
| Auto-detect resolution | `wm size` + desc.txt rewrite at flash time |
| Min API | Android 5.0+ (API 21) |
| Fallback paths | `/data/misc/bootanim/`, `/data/local/`, multiple system paths |
| CP2077 audio pack | 7 OGG files — lock, unlock, charging, camera, UI sounds |
| Per-variant ZIPs | 4 individual variant packages + 1 universal all-in-one |
| Release size | 278 MB universal ZIP + 4 per-variant ZIPs |
